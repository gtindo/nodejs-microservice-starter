# NodeJS Microservice Boilerplate

## Overview

Boilerplate to start a microservice project with nodejs with :
- Express
- MongoDB
- RabbitMQ

## How to install

- Clone repository
- use npm or yarn to install dependencies

````
npm install
````

## How to run project

````
npm start
````

## How to run test
````
npm run test
````
